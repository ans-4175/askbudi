## Install

```sh
$ npm install --save kbbibot
```


## Usage

```js
var kbbibot = require('kbbibot');

kbbibot('Rainbow');
```

```sh
$ npm install --global kbbibot
$ kbbibot --help
```

```sh
# creates a browser.js
$ npm run browser
```
