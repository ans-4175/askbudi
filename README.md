## Install

```sh
$ npm install --save askbudi
```


## Usage

```js
var askbudi = require('askbudi');

askbudi('Rainbow');
```

```sh
$ npm install --global askbudi
$ askbudi --help
```

```sh
# creates a browser.js
$ npm run browser
```
